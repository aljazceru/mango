**iOS -- Create ios/Mango/Mango/MemoryManagementView.swift:**

Follow DocumentLibraryView.swift structure. The view needs:

1. `@EnvironmentObject var appManager: AppManager`
2. `@State private var selectedMemoryId: String? = nil` -- ID of memory being edited
3. `@State private var editText: String = ""` -- current edit text
4. `@State private var memoryToDelete: MemorySummary? = nil` -- for delete confirmation
5. `@State private var showDeleteConfirmation: Bool = false`

Body structure:
- `NavigationStack` containing a `List`
- Back button: `Button("Back") { appManager.dispatch(.popScreen) }` in toolbar leading
- Title: `.navigationTitle("Memories")`
- Empty state: when `appManager.appState.memories.isEmpty`, show `Text("No memories yet. Memories are automatically extracted from your conversations.")` centered
- `ForEach(appManager.appState.memories, id: \.id)` with each row showing:
  - If `selectedMemoryId == memory.id` (edit mode): a `TextField("Memory content", text: $editText, axis: .vertical)` with Save/Cancel buttons (per D-04). Save dispatches `.updateMemory(memoryId: memory.id, content: editText)` and sets `selectedMemoryId = nil`. Cancel sets `selectedMemoryId = nil`.
  - Else (display mode): `VStack(alignment: .leading)` with `Text(memory.contentPreview)` as primary, `Text(memory.conversationTitle ?? "Unknown conversation")` as secondary in `.caption` style, and formatted date. `.onTapGesture` sets `selectedMemoryId = memory.id` and `editText = memory.content` (per D-03).
- `.onDelete` modifier on `ForEach`: sets `memoryToDelete` and `showDeleteConfirmation = true` (per D-05)
- `.confirmationDialog("Delete Memory?", isPresented: $showDeleteConfirmation)` with destructive `Button("Delete")` dispatching `.deleteMemory(memoryId: memoryToDelete.id)` (per D-05)
- `.onAppear { appManager.dispatch(.listMemories) }` to trigger data load (Pitfall 4 mitigation -- though PushScreen::Memories also loads, this is a safety net)

```rust
    Message::OpenMemories => {
        manager.dispatch(AppAction::PushScreen {
            screen: Screen::Memories,
        });
    }
    Message::MemoryStartEdit(id, content) => {
        *memory_edit_state = Some((id, content));
    }
    Message::MemoryEditChanged(text) => {
        if let Some((_, ref mut edit_text)) = memory_edit_state {
            *edit_text = text;
        }
    }
    Message::MemorySaveEdit => {
        if let Some((ref id, ref content)) = memory_edit_state {
            manager.dispatch(AppAction::UpdateMemory {
                memory_id: id.clone(),
                content: content.clone(),
            });
        }
        *memory_edit_state = None;
    }
    Message::MemoryCancelEdit => {
        *memory_edit_state = None;
    }
    Message::MemoryConfirmDelete(id) => {
        manager.dispatch(AppAction::DeleteMemory { memory_id: id });
    }
```
